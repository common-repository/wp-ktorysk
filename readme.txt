WP plugin ktory.sk enables to share question from http://ktory.sk/ 

== Description ==
[ktory.sk]([http://ktory.sk/) is Slovak discussion forum connected to Facebook. You can put there a question started with **`ktory`** (means `which` in Slovak). Whoever can put an answer, and people can vote for that. To every question there is discussion with FB Comment Plugin.

Every question can be **shared**, and for this you can use this WP-plugin.

After installation the plugin, you can use **magic strings** [http://ktory.sk/?id=53&width=400&height=400] in your posts, where **`id`** is question id from URL, and **`width`**, **`height`** are optional parameters, default are 300x300

Please visit [ktory.sk]([http://ktory.sk/) to see that in action.

== Installation ==

1. Upload to your plugins folder, usually `wp-content/plugins/` and unzip the file, it will create a `wp-content/plugins/wp-ktorysk/` directory.

2. Activate the plugin on the plugin screen.

3. Write **magic string** [http://ktory.sk/?id=10&width=300&height=400] into your content of page/post, where **`id`** is question id from URL, and **`width`**, **`height`** are optional parameters, default are 300x300

== Manual Installation ==

In case you don't want to make `wp-config.php` writeable by the server, or you want to make all steps manually:


1. Upload to your plugins folder, usually `wp-content/plugins/` and unzip the file, it will create a `wp-content/plugins/wp-ktorysk/` directory.
